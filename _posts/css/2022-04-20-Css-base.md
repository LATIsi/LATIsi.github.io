## Css (Cascading style sheets)

html에서 꼭 필요한 디자인을 담당하는 스타일 시트 언어
폭포가 되어 떨어지다, 즉 우선순위 순차적용이 폭포가 떨어지듯, 위 - 아래로 아래가 우선순위가 높다.

| css 요소          |                                     설명                                     |
| ----------------- | :--------------------------------------------------------------------------: |
| `selector 선택자` | 스타일을 선택할 특정 html요소 <br> `,` 로 선택자를 여러개 한꺼번에 지정가능! |
| `property 속성`   |                      지정이나 변경하고싶은 속성의 이름                       |
| `value 값`        |                     수치, 특정단위, css에 정의된 키워드                      |

대충

```
selector{
    property :value;
    property2 :value2;
}
```

이런식으로, 선언블록으로 한덩어리를 가지고 세미콜론으로 구분한다.  
주석은 /_ 주석내용 _/ 으로 표시한다.

# 선택자에 대해 깊숙히 들어가자면...

선택자는 총 3가지로 나뉠수 있다.

1. 단순 선택자
2. 복합 선택자
3. 가상 선택자

<br>

---

<br>

## 단순 선택자란?

type 선택자(html 태그), id( # -중복x), class선택자(. -중복 O), 전체선택자 (\*), 속성선택자등이 있다.

속성 선택자일시

```
[속성이름="속성값"] {

}
```

으로 선언해준다. 즉, 특정 html의 속성을 가진 모든요소를 선택한다.  
또한 html에서 클래스 지정시, 띄어쓰기로 여러개 지정가능하다.

```
<h1 class= "class1 class2 class3"></h1>
```

여러개 지정시 바꾸는게 겹치는 속성이 있다면 맨 뒤에 온것으로 적용된다.

## 복합(결합) 선택자란?

<br>
<br>

### 1. 자식 선택자 (child selector)

> <br>
> `>` 을 사용하며, 선택자 a의 모든 자식들중 바로 하위의 자식요소에서 선택자 b와 맞는 요소를 선택.
>
> <br>
>
> ```
> div(선택자a) > p(선택자b) {스타일;}
> ```
>
> <br>

 <br>

### 2. 후손(자손) 선택자 (descendant selector)

> <br>
> `공백`을 사용하며, 선택자a 의 모든 하위 요소(후손) 중에서 선택자b와 맞는요소 모두 선택.
>
> <br>
>
> ```
> div(선택자a)  p(선택자b) {스타일;}
> ```
>
> <br>

<br>

그 외에도  
A+B (형제자매중 a 뒤에 `바로 b가 올때의 선택자`)  
A~B (일반 형제 자매 선택자 요소 뒤에 오는것을 선택.)  
등등이 있다.

## 가상클래스(pseudo-class) 선택자란?

직접 정의할수도 없고 실제 html에 안보인다.  
`선택자 : 가상클래스 {} ` 형식으로 작성된다. 대표적으로는 링크와 관련된 가상 클래스가 있다.

<br>

| 가상클래스 요소            |                                                                            설명                                                                            |
| -------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------: |
| `:link`                    |                                                                  아직 방문하지 않은 링크                                                                   |
| `:visited`                 |                                                                        방문한 링크                                                                         |
| `:hover`                   |                                                                 해당 요소에 커서를 올릴 때                                                                 |
| `:active`                  |                                                            해당 요소가 활성화 / 클릭 되었을 때                                                             |
| `:focus`                   |                                                          해당 요소에 키보드 포커스가 맞춰졌을 때                                                           |
| `:empty`                   |                                                               자식요소가 `없는` 태그에 적용                                                                |
| `:first-child`             |                                                       형제 요소 중 `첫 번째 자식요소(맨 앞)`에 적용                                                        |
| `:last-child`              |                                                           형제 요소 중 `가장 마지막 요소`에 적용                                                           |
| `:nth-child(n)`            |                                      n번째 요소에 스타일 적용, `odd(홀) / even(짝)(2n) / 2n+1` 등의 값으로 사용가능.                                       |
| `:nth-last-child(n)`       | 제일 마지막 요소부터 세며, `마지막에서 n번째에 있는 요소`를 선택. <br> n이 1이면 마지막 요소 선택. `odd(홀) / even(짝)(2n) / 2n+1` 식으로도 사용 가능하다. |
| `타입:nth-of-type(n)`      |                            해당요소의 `특정 타입`을 찾아 순번을 적용한다.<br> 모든 요소를 포함해서 세는 `nth-child`과는 다르다.                            |
| `타입:first-of-type(n)`    |    형제 요소 중 `첫 번째 자식요소(맨 앞)`에 적용하는 :first-child 과는 다르게, <Br> :first-of-type는 형제 요소중 해당 타입의 `첫번째 요소`에 적용한다.     |
| `:only-child`              |                                                         형제 요소 중 `유일하게 하나인` 요소에 적용                                                         |
| `요소1 요소2:only-of-type` |                                요소1중에 딱 하나의 요소2를 가진 요소 선택 - 즉 다른요소내에서 해당유형의 유일한 요소에 적용                                |
| `:not(요소)`               |                                 요소와 일치하지 않는 모든 요소 선택. () 안에는 class나 id, 또는 가상 클래스가 올수도 있다.                                 |

> ## `가상 요소 (Pseudo elements)`
>
> 키워드이며, 콜론 두 개(::)로 시작
>
> <br>

> <br>
> content에 값을 쓰면 그 값이 노출되니 숨기기위해 content : " " 이런식으로 작성하기도 한다.
>
> <br>
>
> > <br>
> >
> > `class요소나 태그 :: before `  
> > 요소의 이전에 들어가는것이 아닌 요소의 `컨텐츠 이전, 즉 지정된 요소의 앞`에 들어가는것.
> > <br>
> >
> > `class요소나 태그 :: after `
> >
> > 지정된 요소의 뒤에 가상요소를 생성시킨다.

# 속성 선택

<br>

| 속성 선택 방법                                |                                                        설명                                                         |
| --------------------------------------------- | :-----------------------------------------------------------------------------------------------------------------: |
| `[속성] / a[href]{ } `                        |                      `a 태그에 src 속성값이 있는것 전부 선택` <br> 특정 속성을 전부 선택한다.                       |
| `[속성 = "값"] / a[src="aaa.png"]{ } `        |           ` a 태그의 src 속성값이 aaa.png인것 전부 선택` <br> 특정 속성의 값을 가진것들을 전부 선택한다.            |
| `[속성^ = "값"] / .toy[category^="swim"]{ }`  |     ` category속성의 swim으로 시작하는값 모두 선택` <br> 특정 문자로 `시작하는` 속성값이 있는 모든요소를 선택.      |
| `[속성 $ = "값"] / img[src $ = ".jpg"]{ } `   |       ` img태그의 src 속성의 값중, .jpg 가 들어간 img를 전부 선택` <br> 속성에 `값`이 포함된 모든요소를 선택.       |
| `[속성 * = "값"] / [.lala * = "heading"]{ } ` | ` lala의 이름을 가진 class의 값에 heading이 들어가 있는 모든 요소 선택.` <br> 특정한 문자를 포함한 모든요소를 선택. |
| `$과 *의 차이`                                |            $는 `접미사- xxx_값, xxx-값`, \*는 `속성명 = "특정 문자가 들어간 속성값 전부ok"`의 차이이다.             |

# css 단위

| 단위                             |                                                                                              설명                                                                                              |
| -------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------: |
| `px (pixel) 화소`                |                                                                         기종의 화소단위 - 모니터에 따라 상대적인 크기                                                                          |
| `pt (point)`                     |                                                                                 문자크기 지정시 사용. 절대크기                                                                                 |
| `in (인치)`                      |                                                                                          인치, 2.54cm                                                                                          |
| `cm`                             |                                                                                            센치미터                                                                                            |
| `mm`                             |                                                                                            밀리미터                                                                                            |
| `em`                             |                                                            부모의 font-size를 1em으로 놓고, 그에 대한 상대적인 크기 지정(배수단뒤)                                                             |
| `rem`                            | em와 비슷하지만 rootme - 최상위 요소의 글자크기를 기준으로 배수로 계산한다.<br> ` n rem =최상위요소 * n = (최상위요소 * n )px`<br> 대부분 rem을 사용시, 루트요소 폰트사이즈를 10px로 지정한다. |
| `%`                              |                                                                백분율 단위, 기본 크기가 100%이고 그에 대한 상대적인 크기를 설정                                                                |
| `vw, vh (viewwidth, viewheight)` |                                                                     화면비 백분율을 지정해 상대적으로 설정한다. (가로세로)                                                                     |
| `vmin`                           |                                                                                          화면 최소값                                                                                           |
| `vmax`                           |                                                                                          화면 최대값                                                                                           |
| `ex`                             |                                                                                    현재 폰트의 x-height 값                                                                                     |
| `svh / svw`                      |                                                          `Short viewport` , 모든 UA 인터페이스(주소 표시줄 포함)를 가정한 뷰포트 크기                                                          |
| `lvh / lvw`                      |                                                           `Large viewport` , 동적으로 늘어나거나 접히는 인터페이스를 가정한 뷰 포트                                                            |
| `dvh / dvw`                      |                `Dynamic viewport` , 모든 UA 인터페이스를 동적으로 고려하는 뷰포트 크기. 인테페이스 요소에 따라 자동으로 조정, <br> `100lvh(최대) 및 100svh(최소)` 의 안에 있다.                |

<br>

### svh / svw 부터는 2022년에 새로 생긴 단위이다.

<br>

<br>

일반적으로 웹의 기본 폰트단위가 `16px`이며..  
특정 div의 부모요소가 16px, 루트요소(html)가 10px일시  
즉 `1rem = 20px / 2em =32px`

기종간, 플랫폼간의 호환성 유지 - 상대적 길이를 사용할시, `px, em과 rem`을 권장한다. / 상소관련으로 rem을 더 권장.

  }
